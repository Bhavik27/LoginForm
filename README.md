# LoginForm
LoginForm using nodeJs

## requirements
- nodejs
- mssql
- express
- body-parser
- connect-flash
- express-session
- express-ejs-layouts

## steps
1. `npm install` to install all dependencies needed.
2. run `db.sql` on your MS Server Management Studio
3. run `nodemon` or `node .\app.js` to start a web.
